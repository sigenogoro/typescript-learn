# 03 - クラスとインターフェース

OOP（オブジェクト指向）の考え方を TypeScript で学ぶ。NestJS のサービス・コントローラはすべてクラスで書くため、ここは特に重要。

## 学習内容

- **クラス** — `class` 定義、コンストラクタ、メソッド
- **アクセス修飾子** — `public` / `private` / `protected` / `readonly`
- **インターフェース** — クラスが満たすべき契約（`implements`）
- **継承** — `extends` で親クラスを引き継ぐ
- **抽象クラス** — `abstract` で直接インスタンス化できないクラスを定義する
- **static メンバー** — インスタンスではなくクラス自体に属するプロパティ・メソッド

## 学習のポイント

- `interface` は型の形だけを定義する（実装を持てない）
- `abstract class` は共通の実装を持ちながら、一部をサブクラスに委譲できる
- NestJS では `@Injectable()` などのデコレータをクラスに付けて DI する
- `private` なプロパティはコンストラクタ引数の前に書く省略記法が使える（`constructor(private name: string)`）
