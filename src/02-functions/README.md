# 02 - 関数

TypeScript における関数の型付けを学ぶ。

## 学習内容

- **関数の型注釈** — 引数と戻り値に型を付ける
- **オプショナル引数** — `?` で省略可能にする
- **デフォルト引数** — 省略時のデフォルト値を設定する
- **Rest 引数** — 可変長引数（`...args`）
- **関数型** — 関数そのものを型として扱う
- **オーバーロード** — 引数の型によって異なる戻り値型を定義する

## 学習のポイント

- `void` は「戻り値がない」ことを表す
- `never` は「絶対に値を返さない」関数（例外を投げるだけ、無限ループ）に使う
- アロー関数も通常の関数と同じように型注釈を付けられる
- 関数型を使うと「コールバック」を型安全に扱える
