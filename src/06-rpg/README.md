# 06 - CLI RPG（学習の集大成）

01〜05 で学んだ TypeScript の知識をすべて活用して、コマンドラインで動く RPG を作る。

## ゲームの仕様

- **キャラクター作成** — 名前・職業（warrior / mage / rogue）を選択
- **ダンジョン探索** — ランダムなエンカウントで敵が出現
- **ターン制戦闘** — 攻撃・魔法・アイテム・逃げるの選択
- **レベルアップ** — 経験値を貯めて能力値が上昇
- **ボス戦** — 最後にボスを倒してクリア

## 使用技術の対応

| 章 | 活用箇所 |
|---|---|
| 01 基本型 | キャラクターのステータス、enum でジョブ管理 |
| 02 関数 | 戦闘計算、アイテム効果など各種ロジック |
| 03 クラス | `Character`, `Enemy`, `Item` クラス |
| 04 ジェネリクス | インベントリ管理、汎用リポジトリ |
| 05 高度な型 | DTO パターン、Utility Types で状態管理 |

## 予定ディレクトリ構成

```
06-rpg/
  index.ts          エントリーポイント・ゲームループ
  models/
    character.ts    プレイヤーキャラクター
    enemy.ts        敵
    item.ts         アイテム
  systems/
    combat.ts       戦闘システム
    inventory.ts    インベントリ
  ui/
    display.ts      画面表示・inquirer 連携
```

## 実行方法（実装後）

```bash
npx tsx src/06-rpg/index.ts
```
